# App to add the checks to linting commits

[![](https://img.shields.io/github/license/monclubsportif/mcs-github-app)](./LICENSE)

## Description

Transfers the checks of unlinted commits to linted commits

## Installation

```bash
$ npm install

$ cp .env.sample .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
